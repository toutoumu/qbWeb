在手机上管理你的 qBittorrent Web UI

QBittorrent Client 是一款第三方移动端客户端工具，可连接你自行部署的 qBittorrent Web UI，随时查看下载状态、添加任务并管理远程下载。

跳转到 App Store 查看隐私协议

本应用不包含下载服务本身，需配合你自己的 qBittorrent Web UI 使用。本应用不是 qBittorrent 官方应用。

核心功能

围绕 qBittorrent Web UI 的常用远程管理场景设计，适合 NAS、家庭服务器、VPS 和个人下载环境。

远程连接

连接你已启用的 qBittorrent Web UI，集中管理不同服务器上的下载任务。

任务管理

查看下载、上传、暂停、完成等状态，并支持暂停、继续、删除等常用操作。

添加下载

支持添加磁力链接和种子任务，让移动端发起远程下载更顺手。

进度与速度

查看实时速度、进度、大小、剩余时间等关键信息，下载状态一眼清楚。

分类与标签

配合 qBittorrent 的分类、标签能力整理任务，适合长期使用的下载库。

本地优先

服务器地址与登录信息用于连接你的 Web UI，开发者无法访问你的下载内容。

应用截图

任务列表、添加任务、设置与详情页面等常用界面预览。
