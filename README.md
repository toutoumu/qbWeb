# QBittorrent Client Web

[https://toutoumu.github.io/qbWeb/](https://toutoumu.github.io/qbWeb/)

## 项目说明

本项目是 **QBittorrent Client** 移动端应用的展示页面与技术支持网站，主要用于 App Store 审核及用户查阅相关文档（如隐私政策、用户协议）。

### 关于 QBittorrent Client

**QBittorrent Client** 是一款第三方移动端客户端工具，专为连接和管理用户自行部署的 qBittorrent Web UI 而设计。它适合在 NAS、家庭服务器、VPS 等环境下管理远程下载任务。

### 核心功能

- **远程连接**：连接并管理不同服务器上的 qBittorrent 实例。
- **任务管理**：查看下载、上传、暂停、完成等状态，支持远程操作。
- **添加下载**：支持磁力链接和种子任务添加。
- **实时监控**：查看实时下载速度、进度、剩余时间等关键信息。
- **分类整理**：配合 qBittorrent 的分类与标签能力高效整理任务。
- **隐私保护**：所有连接信息仅保存在本地，不经过第三方服务器。

### 项目结构

- `index.html`: 项目主页及技术支持页面。
- `privacy.html`: 隐私政策说明页面。
- `app-ads.txt`: 移动应用广告投放验证文件。这个文件没有被使用,使用到的是 toutoumu.github.io 项目下的同名文件
- `screenshots/`: 应用在各平台的展示截图。
